# .net-workshops
.Net Esprit Workshops
to practice differenct aspects of .net framework  



### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/anasbn3issa/.net-workshops.git
   ```
2. Install NPM packages
   ```sh
   npm install
   ```
3. Run the back-end
  
4. Serve the application
  ```sh
   ng serve
   ```
